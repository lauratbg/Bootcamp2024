Ejemplo 1

Si tengo:

a = 2
b = '23'
c = a + +b;

Pero si c = a + b

c entonces es 223

Ejemplo 2

t = [10, 20, 30]

Si se hace un for(i in t) y se saca por consola los valores de 'i' -> se printean los índices (0 1 2)

t[7] = "70"

Ahora se printearía 0 1 2 7

t.push("último elemento")

Ahora se printearía 0 1 2 7 8

Para que por consola salieran los elementos hay que usar:

for(i=0; i < 10; i++) console.log("elementos -> " + t[i]);
